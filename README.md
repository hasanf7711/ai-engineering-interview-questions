## 🧭 Main topics covered

### 🤖 AI Agents

Learn the basics of agents, how they work, and how they make decisions. This section helps with questions about agent loops, tools, planning, and action steps.

### 🧠 LLMs

Review large language models, how they process text, and how they are used in products. This helps with questions about prompts, context windows, and response generation.

### 🔎 RAG

Study retrieval-augmented generation and how it combines search with model output. This is useful for questions about knowledge bases, embeddings, and document search.

### 🛠️ Fine-tuning

Understand when to fine-tune a model and when to use prompt changes instead. You will also see questions about training data, cost, and model fit.

### ⚙️ Quantization

Get a simple view of how model size can be reduced for faster use and lower cost. This section helps with questions about memory, speed, and accuracy trade-offs.

### 🔌 MCP

Learn the basics of model context protocol and how tools connect to AI systems. This is useful for modern AI app and agent questions.

### 🎯 Interview Questions

Use the question and answer format to test your memory and check weak spots before an interview.
